Javascript Functions:
setCookie(cname, cvalue, exDays) - sets a cookie with the given parameters (taken from CS4082 Lab Week 08)
getCookie(cname) - returns the cookie with the given name (taken from CS4082 Lab Week 08)
checkCookies() - onload function that checks if cookies are set yet (taken from CS4082 Lab Week 08 and edited)
rollOver(choice) - this was used for showing the paragraph that I have typed out for each portfolio piece. The choice is either 1, 2, or 3 and depending on which one we have certain divs will get replaced with paragraphs or disapear entirely
save() - this was used to save the text that disappears or is written over in the rollOver() so I can rewrite it when the mouse goes off of it.
mouseOut() - this uses the variables stored in the save() function to restore the original look of the divs after the mouse rolls out of an image of a portfolio piece.
surpiseMe() - this function will select a random one of my webpages to display for 30 seconds. It will then selected another random one to show and will do this until the user stops it.
week8-5.js contains all functions provided to us (with the edits I had to make per the assignment requirements.)
